The SQL Framework written in java.

Parses inputs from the user to:

Create tables of data from files or from user input.
Load Tables.
Insert rows into tables.
Join tables.
Select data from table(s) with "from", "and", and "where" parameters.
Do arithmatic with tables. i.e. to each other or with primitive types.

Tables are composed of rows which are composed of elements. Elements can be of
type int, string, float, NaN (division by zero error element), and NOVALUE.
